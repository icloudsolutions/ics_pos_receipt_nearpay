ICS POS Receipt NearPay

Native NearPay Transaction Details in Standard Odoo 18 POS Receipt

Version 18.0.1.0.0

📋
Overview

The ICS POS Receipt NearPay module extends the standard Odoo 18 Point of Sale receipt to display NearPay transaction details directly in the receipt footer. This module requires the ics_pos_nearpay_integration module to be installed, as it relies on NearPay transaction data stored by that integration.

Based on the official NearPay SDK TransactionReceipt model, this module displays key fields like Approval Code, RRN, Terminal ID, and more — all within Odoo's native receipt system.

🔗
Dependency Requirement

⚠️ Mandatory Dependency

This module depends on ics_pos_nearpay_integration. You must install and configure that module first to capture NearPay transaction data (e.g., nearpay_approval_code, nearpay_retrieval_ref_no, etc.) on pos.payment records.

Requirements

  • Odoo 18.0 with Point of Sale installed
  • ics_pos_nearpay_integration module installed and configured
  • NearPay payment method with payment_method_type = 'terminal'
  • NearPay transaction data stored on pos.payment records (e.g., nearpay_approval_code, nearpay_transaction_id, etc.)

⚙️
Installation & Configuration

  1. Install ics_pos_nearpay_integration first
  2. Download and place ics_pos_receipt_nearpay in your addons folder
  3. Update apps list and install "ICS POS Receipt NearPay"
  4. Go to POS > Configuration > Settings
  5. Enable "Show NearPay Info on Receipt"
  6. Process a NearPay payment — details appear in receipt footer

Supported NearPay Fields (from SDK):

# Mapped from NearPay SDK TransactionReceipt: - nearpay_approval_code → Approval Code - nearpay_retrieval_ref_no → RRN (Retrieval Reference Number) - nearpay_terminal_id → Terminal ID (tid) - nearpay_transaction_id → Transaction UUID - nearpay_card_scheme → Card Network (Visa, Mada, etc.) - nearpay_status_message → Transaction Status - nearpay_currency → Currency (SAR) - nearpay_amount_authorized → Authorized Amount
